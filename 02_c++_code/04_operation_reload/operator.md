# 运算符重载
在C语言中，运算符只能用于基本数据类型，例如，可以用==判断两个整数是否相等，但不能用于判断字符串是否相等，不能用于判断结构体，也不能用于判断类。

在C++中，运算符的重载就是把运算符的符号赋予新的含义，如果类重载了==运算符，那么就可以用==判断两个类是否相等。
